## 📋 Sobre

Repositório de **experimentos e projetos** utilizando **Large Language Models (LLMs)**. Explora técnicas como prompt engineering, fine-tuning, RAG (Retrieval Augmented Generation) e integração com APIs de IA.

## 📁 Estrutura

Os notebooks estão organizados por tema e nível de complexidade, permitindo aprendizado progressivo.

## 🔧 Técnicas Exploradas

- 📝 Prompt Engineering
- 🔗 Chaining de prompts
- 📚 RAG (Retrieval Augmented Generation)
- 🤖 Agentes autônomos
- 🎯 Fine-tuning de modelos
- 🔌 Integração com APIs (OpenAI, Gemini, etc.)

## 📦 Instalação

```bash
git clone https://github.com/LuigiNeto01/projeto_llm.git
cd projeto_llm

pip install -r requirements.txt
jupyter notebook
```
